```md
Create the master mobile UI design style reference for Neo.

App context:
Neo is an AI-powered WhatsApp commerce operating system for Nigerian SMEs. It helps WhatsApp-first sellers reply to customers, capture orders, review AI draft replies, verify manual receipt screenshots safely, manage follow-ups, remember customers, and stay in control from one mobile-first command center.

Target user:
Nigerian WhatsApp-first SME owners and trusted staff who sell through informal chats, voice notes, product screenshots, bank transfers, delivery coordination, and polite local customer language.

Core outcome:
Help a busy seller quickly see urgent work, review customer conversations, approve or edit AI suggestions, capture orders, handle receipt review carefully, send respectful follow-ups, and maintain trust without turning the app into a compressed desktop dashboard.

Visual direction:
Premium, calm, wise, trustworthy, and warmly practical. The app should feel human-first and business-serious, like a competent operations partner. Use warm ivory backgrounds, soft white cards, deep forest green primary actions, subtle sand borders, muted gold highlights, terracotta warmth, charcoal text, and clear status colors. Avoid neon AI visuals, robots, crypto-style dashboards, purple/blue AI gradients, green-only layouts, loud stereotypes, bokeh/orb decoration, and anything that suggests AI replaces human judgment.

UI style guide:
- Background: warm ivory `#FBF7EF`.
- Surface cards and sheets: soft white `#FFFDF8`.
- Alternate quiet panels: soft beige `#F1E7D6`.
- Primary action and active nav: deep forest green `#0E3B2E`.
- Pressed/high-emphasis green: `#092A21`.
- Primary text: charcoal `#1F2522`.
- Muted text: warm gray `#6F766F`.
- Borders and dividers: sand `#E4D8C5`.
- Success: verified green `#247A52`.
- Warning: amber ochre `#B7791F`.
- Error: brick red `#B9473A`.
- Info: muted blue `#2F6F8F`.
- Premium accent: muted gold `#C6A15B`.
- Warm accent: terracotta `#B86A4B`.
- Typography: humanist, editorial, readable, professional; compact screen titles; clear 15px-style body text; tabular numbers for metrics and currency.
- Cards: border-first, 8px maximum radius, 14px to 16px padding, subtle warm shadow only where needed.
- Buttons: 8px radius, 48px default height, strong primary fill, clear secondary outline style.
- Inputs: 8px radius, sand border, readable labels, short helper text.
- Icons: rounded outline icons, medium stroke, familiar symbols for send, edit, check, x, search, filter, bell, receipt, message, package, user, settings.
- Navigation: stable bottom tabs for Today, Inbox, Approvals, Follow-ups, and Settings.
- Status badges: pill-shaped with color plus label/icon, never color alone.

Create:
A high-fidelity mobile app UI style reference showing the overall design language for Neo.

Include:
- Example compact header with app title and connection/status control.
- Example operational card.
- Example primary button.
- Example secondary button.
- Example bottom navigation.
- Example empty state block.
- Example form field.
- Example list item.
- Example status chip or badge.
- Example rounded outline icon style.
- Example AI draft or receipt review decision card that clearly keeps a human in control.

Requirements:
- Mobile-first.
- Portrait mobile aspect ratio around 390x844 or 430x932.
- Single mobile UI style board.
- Clean app screenshot style.
- Premium production-ready interface.
- Consistent spacing on a 4px grid.
- Consistent 8px card/input/button radius.
- Consistent subtle border and shadow treatment.
- Consistent typography hierarchy.
- Consistent color palette.
- No browser frame.
- No phone frame.
- No fake desktop UI.
- No random neon gradients unless replaced by the Neo palette.

Text rendering rule:
Use clear readable UI labels where possible. Avoid tiny paragraphs. Use realistic short text blocks and labels. Exact final copy will be implemented in code, so prioritize layout accuracy, hierarchy, and component structure over perfect text rendering.

Avoid:
- Robots, generic AI brains, magic wands, sparkles-as-AI, and holograms.
- Purple/blue AI gradients.
- Black-glass dashboards.
- Overstuffed dashboard widgets.
- Desktop tables compressed into mobile.
- Nested cards.
- Large bubbly cards.
- Lifestyle phone-in-hand mockups.
- Fake browser chrome.
- Any visual implication that manual transfer screenshots are automatically confirmed.

Output:
A single mobile UI style board in portrait format that can be used as the visual reference for all future Neo screen designs.

Save generated image as:
design-assets/ui-screens/master-ui-style-reference.png
```
