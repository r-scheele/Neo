# App Store Screenshot Prompt

Use this only after real app screens exist. Store screenshots should be composed from real screen captures, not imagined UI.

```md
Create app store screenshot artwork for Neo using the provided real app screenshots.

App context:
Neo is a mobile-first AI-powered WhatsApp commerce operating system for Nigerian SMEs. It helps sellers reply customers, capture orders, verify payments, review receipts safely, coordinate delivery, manage workflows, remember customers, recover sales, and keep daily WhatsApp commerce operations organized from their phone.

Real app screenshots provided:
- Setup Checklist screen
- Today Command Center screen
- Inbox screen
- Conversation Detail with AI Draft screen
- Receipt Review screen
- Follow-ups screen

Visual direction:
Use Neo's approved visual direction: premium, calm, wise, trustworthy, warmly practical. Use deep forest green, warm ivory, soft beige, muted gold, terracotta warmth, and charcoal. Use subtle editorial commerce motifs, soft shadows, and restrained composition.

Composition:
Create a cohesive store screenshot set that places the real screenshots in clean device frames over warm ivory or deep forest green branded backgrounds. Add only subtle supporting shapes from the approved style anchor: chat bubble, receipt slip, product card, delivery pin, follow-up reminder, and shield/check motif.

Requirements:
- Use the real app screenshots as the central content.
- Do not alter the UI inside screenshots except for safe cropping, masking, or device framing.
- Keep every screenshot readable at store preview size.
- Use text only if exact marketing captions are provided separately. If no captions are provided, produce caption-free artwork.
- Keep backgrounds calm and uncluttered.
- Respect App Store and Google Play screenshot rules.
- Make the set feel consistent across all frames.

Avoid:
- No fake app UI.
- No inaccurate feature claims.
- No tiny unreadable text.
- No competitor references.
- No WhatsApp logo imitation.
- No robots, neon AI, sparkles, holograms, or generic AI brain imagery.
- No busy marketplace scenes or loud decorative stereotypes.

Output:
Create a screenshot artwork set named store-screenshot-set-neo.png for review.
If platform-specific exports are supported later, prepare iPhone portrait and Android portrait variants using the same composition system.

Acceptance check:
The screenshots should honestly show the app, communicate value quickly, and feel like Neo: calm control over WhatsApp commerce.
```
