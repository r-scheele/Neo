# Empty State Prompts

Use these when a screen has no user data yet. Empty states should feel intentional and helpful, not shameful.

## Empty Inbox

```md
Create an empty-state illustration for the Inbox screen in Neo.

App context:
Neo is a mobile-first AI-powered WhatsApp commerce operating system for Nigerian SMEs.

Empty state context:
The user sees this when no WhatsApp conversations have arrived yet or when WhatsApp setup has just been completed. The desired next action is to test the WhatsApp connection or wait for the first customer message.

Style reference:
Use the approved Neo style anchor for palette, lighting, soft geometry, and premium editorial flat illustration.

Concept:
A calm empty inbox tray with one soft chat bubble outline arriving gently, plus a small connection/check motif. No readable text.

Requirements:
- Transparent background.
- No text, letters, or numbers.
- Warm and reassuring.
- Clear at small mobile sizes.
- Leaves room for headline, body copy, and button.

Avoid:
- No sad imagery.
- No fake WhatsApp logo.
- No busy chat list.
- No robot assistant.

Output:
Save as empty-inbox.png.
```
