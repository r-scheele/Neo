# Onboarding Hero Prompt

```md
Create an onboarding hero illustration for Neo, a mobile-first AI-powered WhatsApp commerce operating system for Nigerian SMEs.

App context:
Neo helps WhatsApp-first sellers reply customers, capture orders, verify payments, review receipts safely, coordinate delivery, remember customers, manage workflows, recover sales, and keep customer operations organized from their phone.

Style reference:
Use the approved Neo style anchor. If the approved logo mark or brand companion object is available, use it only as a style and motif reference. Preserve palette, proportions, lighting, and premium editorial flat style.

Scene:
Show a calm mobile commerce operating system coming into order: chat bubbles flowing into organized order cards, a receipt being reviewed with a visible check/shield motif, a product card, a customer memory card, a workflow/job card, a delivery pin, and a follow-up reminder card. The objects should feel like WhatsApp commerce tasks becoming one connected system, but there must be no readable UI text.

Requirements:
- Mobile portrait friendly.
- Clear central focal point.
- No embedded text, letters, numbers, or fake readable messages.
- Leave open space at the bottom for onboarding copy and a button.
- Works on a warm ivory background.
- Shows the emotional promise: calm control over busy WhatsApp selling.
- Avoids detailed fake app screens.
- Composition remains readable when cropped on different phone sizes.

Avoid:
- No clutter.
- No robots, neon AI, holograms, sparkles, or brain imagery.
- No screenshots of WhatsApp or competitor apps.
- No stereotypical cultural decoration.
- No payment confirmation visual that looks automatic.

Output:
High-resolution portrait image, transparent background if possible.
Save as onboarding-hero-neo-operating-system.png.

Acceptance check:
The image should make Neo's promise understandable in one glance: messy chats, orders, receipts, and follow-ups becoming calm and organized.
```
