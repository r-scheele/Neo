# Splash Screen Prompt

```md
Create a splash screen visual for Neo, a mobile-first AI-powered WhatsApp commerce operating system for Nigerian SMEs.

App context:
Neo helps sellers reply customers, capture orders, verify payments, review receipts safely, coordinate delivery, manage workflows, remember customers, recover sales, and keep daily WhatsApp commerce operations organized from their phone.

Style reference:
Use the approved Neo logo mark and style anchor. Preserve palette, simplicity, geometry, and calm premium mood.

Splash direction:
Centered logo mark on a clean branded background. Use either deep forest green background with warm ivory logo contrast or warm ivory background with forest green logo, choosing the option with stronger small-screen clarity.

Requirements:
- Mobile portrait friendly.
- Clean centered composition.
- No text unless a separate approved wordmark is provided later.
- No letters, numbers, or readable UI.
- Safe margins for different device sizes.
- High contrast.
- No small details near screen edges.
- No complex scene or fake app screen.

Avoid:
- No animation frames.
- No robots, neon AI, holograms, or sparkles.
- No busy illustration.
- No transparent-only output if a solid launch background is required.

Output:
Save as splash-screen-neo.png.
Also provide the background color value used.

Acceptance check:
The splash should load cleanly, feel branded, and transition naturally into Neo's warm ivory app interface.
```
