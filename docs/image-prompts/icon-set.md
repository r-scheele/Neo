# Icon Set Prompt

Use this for custom Neo category/status icons. For standard UI actions during implementation, prefer a proven icon library where available.

```md
Create a consistent custom icon set for Neo, a mobile-first AI-powered WhatsApp commerce operating system for Nigerian SMEs.

App context:
Neo helps sellers reply customers, capture orders, verify payments, review receipts safely, coordinate delivery, manage workflows, remember customers, recover sales, and keep daily operations organized from their phone.

Style reference:
Use the approved Neo style anchor for palette, rounded geometry, soft corners, professional warmth, and calm trust-first mood.

Icon style:
Rounded outline icons with medium stroke weight, consistent optical size, consistent padding, and simple shapes readable at 24px.

Icons needed:
- Today: daily command center and urgent work
- Inbox: customer conversations
- Approvals: sensitive AI actions waiting for review
- Follow-ups: respectful reminders and lead recovery
- Settings: business and safety rules
- Receipt Review: manual bank transfer receipt inspection
- Order: structured customer order
- Customer: buyer profile and memory
- Product: catalog item or product card
- Delivery: location and dispatch
- AI Draft: assistant reply draft requiring review
- Warning: needs attention or low confidence
- Paid: verified payment
- Offline: limited connection
- Permission: role-based access

Requirements:
- No text, letters, numbers, or labels inside the icons.
- Transparent background.
- Consistent stroke weight and corner radius.
- Same visual size and padding for every icon.
- Works on light warm ivory and white surfaces.
- Use forest green, charcoal, muted gold, terracotta, warning ochre, and success green only where useful.
- Icons must remain recognizable at 24px.
- Keep the family professional, not playful.

Avoid:
- No mixed 3D and flat styles.
- No filled icons mixed with outline icons unless selected states are provided separately.
- No gradients.
- No robot, brain, sparkle, magic wand, or neon AI symbols.
- No WhatsApp logo or payment-provider logos.
- No complex illustrations.

Output:
Create one high-resolution contact sheet for review named icon-set-neo-core.png.
If individual transparent exports are supported, also export each icon as:
icon-today.png
icon-inbox.png
icon-approvals.png
icon-follow-ups.png
icon-settings.png
icon-receipt-review.png
icon-order.png
icon-customer.png
icon-product.png
icon-delivery.png
icon-ai-draft.png
icon-warning.png
icon-paid.png
icon-offline.png
icon-permission.png

Acceptance check:
The icons should look like one family, feel native to Neo's calm operations product, and remain clear at small mobile UI sizes.
```
