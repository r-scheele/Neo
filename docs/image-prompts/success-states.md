# Success State Prompts

Success states should be positive but restrained. Neo is an operations product, so confirmations should feel reliable rather than flashy.

## Reply Sent

```md
Create a success-state illustration for a reply sent in Neo.

App context:
Neo is a mobile-first AI-powered WhatsApp commerce operating system for Nigerian SMEs.

Success context:
The user sees this after reviewing and sending a customer reply. The desired feeling is calm progress and confidence.

Style reference:
Use the approved Neo style anchor. Preserve palette, soft lighting, premium editorial flat style, and gentle shadows.

Concept:
A chat bubble moving from a draft card into a small sent check motif, with a subtle product-card shape nearby to imply context-aware reply.

Requirements:
- Transparent background.
- No readable message text.
- Positive but quiet.
- Works inside a mobile confirmation or toast-adjacent modal.
- Use success green sparingly.

Avoid:
- No confetti.
- No robot.
- No fake WhatsApp UI.
- No sparkle magic.

Output:
Save as success-reply-sent.png.
```

## Payment Verified

```md
Create a success-state illustration for a human-reviewed payment verified in Neo.

App context:
Neo helps Nigerian WhatsApp sellers review receipt screenshots safely before confirming payment.

Success context:
The user sees this after a permitted user verifies a manual transfer against bank alert and confirms the order payment. The desired feeling is trust and careful completion.

Style reference:
Use the approved Neo style anchor for palette, lighting, shape language, and premium editorial flat illustration.

Concept:
A receipt slip beside a shield/check motif and a small order card. The visual should imply careful human verification has been completed, not that a screenshot was automatically trusted.

Requirements:
- Transparent background.
- No readable receipt text, bank names, or amounts.
- Use success green in a restrained way.
- Calm and trustworthy.

Avoid:
- No "auto approved" feeling.
- No large paid stamp.
- No fake bank interface.
- No money shower.
- No robot or neon AI.

Output:
Save as success-payment-verified.png.
```
