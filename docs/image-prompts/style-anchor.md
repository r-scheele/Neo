# Style Anchor Prompt

Copy-paste this prompt first. Use the approved output as the visual reference for every later Neo asset.

```md
Create the primary visual style anchor for Neo, a mobile-first AI-powered WhatsApp commerce operating system for Nigerian SMEs.

App context:
Neo helps WhatsApp-first business owners and trusted staff reply customers, capture orders, verify payments, review receipt screenshots safely, coordinate delivery, remember customers, manage workflows, recover sales, and keep daily commerce operations organized from their phone.

Target user:
Nigerian SME owners and staff running fashion, tailoring, food, beauty, logistics, real estate, and general WhatsApp-first commerce businesses.

Core mood:
Premium, calm, wise, trustworthy, warmly practical, human-first.

Visual direction:
Use a deep forest green, warm ivory, soft beige, muted gold, terracotta warmth, and charcoal palette.
Use premium editorial flat illustration with softly rounded geometric shapes, subtle warm paper texture, gentle ambient shadows, and clean mobile-friendly proportions.
Use soft lighting from the upper left.
Show the feeling of organized WhatsApp commerce without showing any readable text.

Composition:
Create a square style anchor showing a calm arrangement of abstract commerce objects: a chat bubble shape, a receipt slip, a product card, a payment checkmark, a delivery pin, and a small folded-paper/leaf brand motif. Arrange them like a trustworthy operations system coming into order.

Requirements:
- No text, letters, numbers, or readable UI.
- No brand names or logos.
- No people with detailed faces.
- No robots, neon AI, holograms, sparkles, magic wands, or brain imagery.
- No fake dashboard screenshots.
- No loud cultural stereotypes.
- Clear silhouette at small mobile sizes.
- Balanced spacing with no important detail near the edges.
- High-quality square image.
- Transparent background if possible.

Output:
Save as style-anchor-neo-commerce-system.png.

Acceptance check:
The image must define the palette, proportions, line weight, lighting, object shapes, and calm premium mood clearly enough that future Neo assets can match it.
```
