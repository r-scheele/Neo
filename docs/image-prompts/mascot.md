# Mascot / Brand Companion Prompt

Neo should not use a playful character mascot in the MVP. The brand handoff specifically avoids robot visuals and loud AI imagery. If a reusable companion asset is needed for empty, success, and error states, use a non-character brand companion object instead.

```md
Create a non-character brand companion object for Neo, a mobile-first AI-powered WhatsApp commerce operating system for Nigerian SMEs.

Style reference:
Use the approved Neo style anchor for palette, lighting, proportions, line weight, and premium editorial flat illustration style.

Companion concept:
A small calm guide object made from a folded receipt, a soft chat bubble, and a subtle leaf/check motif. It should feel like a trustworthy business helper, not a creature, not a robot, and not a cartoon mascot.

Personality:
Calm, wise, warm, practical, professional, Nigerian-aware without stereotypical decoration.

Requirements:
- Clear reusable silhouette.
- No face, limbs, robot parts, or animal-like features.
- No text, letters, numbers, or brand names.
- Transparent background.
- Leave comfortable padding around the object.
- Use deep forest green, warm ivory, soft beige, muted gold, terracotta, and charcoal in the same proportions as the style anchor.
- Simple enough to reuse in empty, success, error, and onboarding states.

Avoid:
- No robots.
- No childish character design.
- No neon AI, brain, sparkles, magic wand, or hologram motifs.
- No busy background.
- No props that imply automatic payment confirmation.
- No cultural stereotypes or flags as decoration.

Output:
Save as mascot-neo-guide-object.png.

Acceptance check:
The object should work as a subtle brand companion across states while preserving Neo's serious, trust-first commerce personality.
```
