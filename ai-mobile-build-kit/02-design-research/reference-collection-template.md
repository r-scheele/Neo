## Collection Tips

- Capture the full device frame when reviewing mobile screens.
- Include boring screens like settings, empty states, and error states.
- Save examples of forms and validation, not only hero screens.
- Include at least one app with similar data density.
- Include at least one app with similar emotional tone.

## Design Synthesis

After collecting references, answer:

- What do the best references have in common?
- What is inappropriate for our target user?
- Which visual choices support the product outcome?
- Which patterns are trendy but unnecessary?

## Done Looks Like

The collection is useful when every reference has a purpose and no reference is being used as a clone target.
