# Design Research Checklist

Use this before asking AI to build screens. The goal is not to copy references. The goal is to define a visual direction that makes implementation decisions easier.

## Reference Sources

- Pinterest: mood, visual language, colors, lifestyle imagery, illustration direction.
- Dribbble: polished UI ideas, empty states, icon treatments, card patterns.
- Mobbin: real mobile flows, navigation patterns, onboarding, paywalls, settings, forms.
- Behance: brand systems, typography pairings, campaign visuals, illustration systems.
- Real apps: App Store and Google Play apps that solve adjacent problems.

## Collection Checklist

- [ ] Collect 5 to 12 references.
- [ ] Include at least 3 real app references.
- [ ] Capture screenshots of full screens, not only cropped details.
- [ ] Note what you like about each reference.
- [ ] Note what should not be copied.
- [ ] Identify repeated patterns across references.
- [ ] Save references in a shared folder.

## What To Look For

- Mood: calm, energetic, premium, playful, clinical, editorial, utilitarian.
- Typography: rounded, geometric, humanist, compact, expressive, system-first.
- Color palette: primary, secondary, neutrals, success, warning, error.
- Layout: dense, spacious, card-based, list-based, editorial, dashboard-like.
- Icon style: outline, filled, duotone, hand-drawn, geometric.
- Card style: flat, bordered, elevated, tinted, image-led.
- Illustration style: 3D, flat vector, line art, clay, photographic, mascot-led.
- Shadows: none, subtle, layered, soft ambient.
- Motion: snappy, calm, springy, minimal, celebratory.

## Avoid Copying Pixel-For-Pixel

Do:

- Borrow principles.
- Describe patterns in your own words.
- Combine references into a new direction.
- Use references to define constraints.

Do not:

- Recreate another app's exact layout.
- Copy brand colors, icons, or proprietary illustrations.
- Use screenshots as final app assets.
- Ask AI to clone a competitor.

## Done Looks Like

Design research is done when you can write a visual direction without saying "like this app exactly."
