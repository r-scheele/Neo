# Google Play Submission Checklist

Use this for Android release preparation.

## Play Console

- [ ] App record created.
- [ ] Package name matches build.
- [ ] App category selected.
- [ ] Content rating completed.
- [ ] Data safety form completed.
- [ ] Target audience completed.
- [ ] Privacy policy URL added if required.

## Assets

- [ ] App icon final.
- [ ] Feature graphic prepared.
- [ ] Phone screenshots prepared.
- [ ] Tablet screenshots if applicable.
- [ ] Store copy reviewed.

## Build

- [ ] Android production build created.
- [ ] App bundle uploaded.
- [ ] Internal testing completed.
- [ ] No dev-only features visible.

## Policy

- [ ] Permissions are justified.
- [ ] Data safety answers match app behavior.
- [ ] Account deletion requirements handled if needed.
- [ ] Ads declaration correct.

## Done Looks Like

The Google Play submission is ready when the release build, listing, and data safety form tell the same truth.
