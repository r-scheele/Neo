# App Store Submission Checklist

Use this for iOS release preparation.

## App Store Connect

- [ ] App record created.
- [ ] Bundle ID matches build.
- [ ] App name and subtitle ready.
- [ ] Category selected.
- [ ] Age rating completed.
- [ ] Privacy details completed.
- [ ] Support URL ready.
- [ ] Marketing URL if needed.

## Assets

- [ ] App icon final.
- [ ] Screenshots for required devices.
- [ ] Preview video if used.
- [ ] Store copy reviewed.

## Build

- [ ] Production iOS build uploaded.
- [ ] Build processed successfully.
- [ ] TestFlight smoke test complete.
- [ ] No dev-only features visible.

## Privacy

- [ ] Data collection matches implementation.
- [ ] Analytics disclosed.
- [ ] Account deletion flow or instructions provided if required.
- [ ] Permissions have clear purpose.

## Done Looks Like

The iOS submission is ready when the App Store listing honestly matches the production build and privacy disclosures.
