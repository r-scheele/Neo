# Splash Screen Prompt

Use for a launch screen mark or simple splash visual. Keep it simple because splash screens appear briefly.

```md
Create a splash screen visual for [APP_NAME].

App context:
[APP_NAME] is a [TYPE_OF_APP] for [TARGET_USER] that helps them [CORE_OUTCOME].

Style reference:
Use the approved logo mark and style anchor. Preserve palette, simplicity, and geometry.

Splash direction:
[CENTERED_LOGO / SIMPLE_MARK / SUBTLE_BRAND_SHAPE]

Requirements:
- Clean centered composition.
- Works on mobile portrait screens.
- No text unless the final app name wordmark is explicitly required.
- Safe margins for different device sizes.
- High contrast against the splash background.
- No small details near screen edges.
- Avoid busy illustration.

Avoid:
- No complex scene.
- No fake UI.
- No animated frames unless specifically requested.
- No unreadable text.
- No transparent-only asset if the platform requires a solid background.

Output:
High-resolution centered splash source plus background color value.

Acceptance check:
The splash should load cleanly, feel branded, and not compete with the first real screen.
```
