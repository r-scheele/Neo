# Logo Generation Prompt

Use after the style anchor is approved.

```md
Create a mobile app logo mark for [APP_NAME].

App context:
[APP_NAME] is a [TYPE_OF_APP] for [TARGET_USER] that helps them [CORE_OUTCOME].

Style reference:
Use the approved style anchor for palette, simplicity, geometry, and mood. Do not copy the exact composition.

Logo direction:
[SYMBOL_IDEA, BRAND_PERSONALITY, SHAPE_PREFERENCES]

Requirements:
- Simple geometric mark.
- Recognizable at small app icon sizes.
- No text, letters, or words.
- Works in square app icon format.
- Works on light and dark backgrounds if possible.
- Uses the approved app palette.
- Avoids excessive detail.
- Feels native to a modern mobile app.

Avoid:
- No realistic photos.
- No complex scenes.
- No thin fragile lines.
- No gradients unless included in the approved visual direction.
- No accidental typography.
- No stock-looking generic symbol.

Output:
High-resolution square image. Provide transparent-background version for logo usage and solid-background version if generating an app icon source.

Acceptance check:
The mark should be identifiable at 48px, feel aligned with the style anchor, and not depend on text.
```
