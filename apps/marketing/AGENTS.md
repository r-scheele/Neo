# Marketing Site Instructions

This workspace contains the `neo.com` marketing site.

- Build public, SEO-friendly marketing pages only.
- Keep CTAs pointed at `app.neo.com`, waitlist/signup flows, or mobile app download destinations when available.
- Do not implement authenticated seller operations here.
- Do not import design reference screenshots from `design-assets/` into runtime code.
- Do not use private commerce data, customer names, message text, receipt images, payment proof, exact addresses, tokens, or secrets in analytics or UI fixtures.
- Use only public `NEXT_PUBLIC_*` variables in client-rendered code.
- Keep the visual system aligned with Neo: warm ivory, deep forest green, charcoal text, muted gold/cream accents, calm premium spacing.
